3

📐 Physics-Based Energy Model

Data center energy consumption follows a well-established formula based on electrical engineering principles:

Energy (MWh) = Capacity (MW) × 8,760 hrs × Utilization × PUE
Example: 50 MW × 8,760 × 0.60 × 1.40 = 367,920 MWh/year
Parameter Description Values Used
PUE Power Usage Effectiveness (total power / IT power) 1.10 (hyperscale) — 1.50 (colo)
Utilization Average server load percentage 55% — 70%
8,760 Hours per year (24 × 365) Constant

This yields ~5,600 — 7,300 MWh per MW per year, aligned with industry benchmarks from Google, Microsoft, and DOE reports.

🤖 Machine Learning Adjustments

While the physics model provides the foundation, we train ML models to capture state-specific variations caused by:

  • Climate: Arizona data centers need more cooling than Oregon
  • Grid Efficiency: States with cleaner grids may attract more efficient operators
  • Regulations: California has stricter efficiency standards
  • Data Center Mix: Virginia has more hyperscale facilities
